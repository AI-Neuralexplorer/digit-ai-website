Set up Plausible Analytics

To enable this dashboard:

1. Sign up at plausible.io
2. Add digit-ai.ai as a site
3. Go to Site Settings → Visibility → make the dashboard public or shared
4. Copy the shared dashboard URL and add it to Vercel environment variables as PLAUSIBLE_DASHBOARD_URL
5. The iframe will automatically load once the variable is set
